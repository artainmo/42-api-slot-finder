# Script to find slots

### PREPARE

* Python3 installed

* Make env

### USE
Create access-token
```
./generate_access_token.sh
```
Find slots
```
./slot_find ACCESS_TOKEN
```


### OTHER
* Project not found
```
src/project_notation_find.sh
```
Find the slug value of your project. The script adds 42cursus- before the input you give.
